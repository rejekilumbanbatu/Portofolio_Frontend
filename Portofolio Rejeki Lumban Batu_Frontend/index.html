Proyek Saya

Prediksi Harga Komoditi Jagung

Website untuk memantau harga jagung harian, mingguan, dan tahunan serta menampilkan prediksi harga berdasarkan data historis. Sistem ini membantu petani dan pedagang membuat keputusan jual beli dengan lebih baik.

Teknologi: HTML, CSS, JavaScript, PHP, MySQL

Website Informasi Budaya Suku Jawa

Website statis yang menampilkan sejarah, adat istiadat, kuliner, tempat wisata, dan kalender tradisional Suku Jawa dengan tampilan modern dan responsif.

Teknologi: HTML, CSS, JavaScript, PHP, MySQL

Sistem Pemberi Pakan Ayam Otomatis Berbasis IoT

Sistem IoT untuk memberi pakan ayam secara otomatis dan manual melalui aplikasi Blynk, serta memonitor kondisi pakan dan air dari smartphone.

Teknologi: Arduino, ESP32/8266, Blynk, Sensor Ultrasonik, Relay

Sistem Pengering Tangan Otomatis

Alat pengering tangan otomatis menggunakan sensor ultrasonik untuk mendeteksi tangan dan mengaktifkan kipas melalui relay yang dikendalikan oleh Arduino Mega.

Teknologi: Arduino Mega, Sensor Ultrasonik, Relay, Fan DC

Sistem Pendeteksi Ketinggian Air Danau Toba

Sistem IoT berbasis LoRa untuk memonitor ketinggian air dan kecepatan angin secara real-time menggunakan sensor JSN-SR04T dan anemometer, dengan data dikirim ke dashboard web.

Teknologi: LoRa, ESP32, JSN-SR04T, Anemometer, Panel Surya
